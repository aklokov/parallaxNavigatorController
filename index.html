Parallax UINavigatorController for iPhone

About

it is UI tweak for UINavigationController. Have you see parallax background on Android Main screen. It's the same but for iPhone.

how to use

It's very easy. You can extend your current application by simple steps: 1. put 2 files from 'ParallaxUINavigatorController lib' to your project 2. add to your project image that will be used for Parallax 3. update your applicatio .plist by item "ParallaxBackground" with name of image sfom step 3 4. in storyboar change class name (section 'custom class' in properties window) from UINavigatorController to ParallaxUINavigationController

example

This project contains Example in 'ParallaxUINavigatorController'. You can check functionality from there.
